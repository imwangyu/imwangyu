登录与注销

注销Linux

exit

图形界面与运行级别

X Window

重启X Window

  1. 注销后重新登录
  2. 快捷键：Alt + Ctrl + Backspace

终端界面与图形界面切换

终端界面：Ctrl + Alt + F1~F6，运行级别为3,进程名对应tty1~tty6
图形界面：Ctrl + Alt + F7，运行级别为5, 进程名tty7

运行级别

  • 0：关机
  • 3：完全多用户模式
  • 5：X Window系统
  • 6：重启系统

默认运行级别

/etc/inittab
