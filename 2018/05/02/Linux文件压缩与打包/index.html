Linux文件压缩与打包

文件压缩

压缩比

压缩前与压缩后文件所占用的磁盘空间大小的比。

常用压缩命令与技术

Compress

已经过时

gzip,zcat

文件格式

*.gz

常用命令

  1. gzip
  2. zcat：如果压缩的是文件，可以使用该命令读取

bzip2,bzcat

压缩比优于gzip

文件格式

.bz,.bz2,.tbz,.tbz2

常用命令

  1. bzip2
  2. bzcat：如果压缩的是文件，可以使用该命令读取

tar：打包

通过gzip/bzip2同时进行压缩

参数

  1. -z, –gzip, –gunzip, –ungzip：通过 gzip 过滤归档
  2. -x：解打包或解压缩
  3. -v, –verbose 详细地列出处理的文件
  4. -f：文件名称
  5. -j, –bzip2 通过 bzip2 过滤归档
  6. -p：保存文件原本的权限与属性
  7. -c：新建打包文件
